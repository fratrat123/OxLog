Getting Started

            

                

                    
Version Auto-Increment

                    

                        After you Log & Push, the patch version automatically increments by 1. For example, if you log v1.2.0, the version bumper resets to v1.2.1 ready for your next change. You can still manually bump Major or Minor at any time — the auto-increment only touches the patch number.
                    

                

                

                    
Creating a Discord Webhook

                    

                        In Discord, right-click a channel → Edit Channel → Integrations → Webhooks → New Webhook. Name it something like "OxLog" and copy the webhook URL. Paste it into the Webhook tab for each plugin. You can use one webhook for all plugins or separate webhooks per channel.
                    

                

                

                    
Paste Code Workflow

                    

                        Copy your updated plugin code to the clipboard, click Paste Code, review it, then click Use This Code. A green bar appears confirming the code is staged. When you hit Log & Push, the code is written to the live .cs file, the [Info] version is updated, a snapshot is saved, and the changelog is pushed to Discord — all in one click.
                    

                

                

                    
Snapshots & Reverting

                    

                        Every log entry snapshots the full .cs file along with a notes.txt. In the History tab, click Diff to see exactly what changed, or Revert to roll back. Reverting snapshots the current live file first (so you never lose anything), then restores the selected version.
                    

                

                

                    
RCON Console

                    

                        Set your server's +rcon.password and port in Settings, then click Connect in the Console tab. Oxide auto-compiles when files change, but you can manually Load, Unload, or Reload plugins. Use the filter to show only output from a specific plugin, or click Errors to isolate problems.
                    

                

                

                    
Keyboard Navigation

                    

                        Press any letter key to jump to the first plugin starting with that letter. Press the same letter again to cycle through matches. Works anywhere except when typing in an input field.
                    

                

                

                    
Remote Access & Clipboard

                    

                        If Paste Code shows a clipboard error over Tailscale or LAN, go to chrome://flags/#unsafely-treat-insecure-origin-as-secure, add your OxLog URL (e.g. http://100.x.x.x:5000), enable it, and relaunch Chrome. OxLog will guide you through this automatically if it happens.
